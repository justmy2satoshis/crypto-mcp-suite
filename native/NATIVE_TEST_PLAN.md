2. Navigate to `Crypto-MCP-Suite\native\`

**Expected Results:**

### TC-13: Resource Usage

**Steps:**

3. Measure CPU and memory usage

**Expected Results:**
- ‚úÖ Total memory < 1GB (Essential tier target: ~500MB)

### TC-14: Startup Time

**Objective:** Measure time to start all MCPs

**Steps:**

2. Measure start time

**Expected Results:**
- ‚úÖ All 7 MCPs start within 60 seconds
- ‚úÖ Health checks respond within 30 seconds after start

**Evidence Required:**
```powershell

$startTime = Get-Date
crypto-mcp start essential
Start-Sleep -Seconds 30
$endTime = Get-Date

"Startup time: $duration seconds" > test-results\tc14-timing.txt
pm2 list >> test-results\tc14-timing.txt
```
